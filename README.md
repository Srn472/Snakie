
# Snake Game (Snakie)

A classic timeless game where players control a growing snake, guiding it to eat "food" (planets) while avoiding collisions with walls and obstacles. With each bite, the snake grows longer, which also makes the navigation increasingly tricky.

![Gameplay Screenshot](https://raw.githubusercontent.com/Srn472/Snakie/master/screenshot.png)

## Features

- **Classic Snake Gameplay:** Enjoy the timeless fun of the classic snake game.
- **Space Theme:**行星s and stars create a space-themed environment.
- **Score Tracking:** Keep track of your high scores.
- **Easy Controls:** Simple and intuitive controls for smooth gameplay.

## How to Play

1. Use the arrow keys to move the snake.
2. Eat planets to grow your snake.
3. Avoid hitting walls or running into yourself.
4. Earn points for each planet eaten.

## Getting Started

### Prerequisites

- Python 3.x
- Pygame library (if not installed, you can install it using `pip install pygame`)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Srn472/Snakie.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Snakie
   ```
3. Run the game:
   ```bash
   python snakee.py
   ```

### Game Controls

- **Arrow Keys:** Move the snake in the desired direction.
